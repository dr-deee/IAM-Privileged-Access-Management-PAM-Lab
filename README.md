# Privileged Access Management (PAM) Lab

## Project Overview

This project simulates a Privileged Access Management (PAM) workflow within an Active Directory environment.

The objective was to demonstrate how organizations manage privileged access through controlled request, approval, provisioning, review, and removal processes.

The project follows a complete privileged access lifecycle, ensuring that elevated permissions are granted only when required and removed once no longer needed.
